# neon-aura-ar
Browser-based AR interaction system using hand tracking, voice commands, and real-time visual + audio effects.
# ⚡ NEON AURA AR

> A browser-based **gesture + voice controlled AR interaction system** built using computer vision, real-time rendering, and procedural audio.

---

## 🚀 Overview

NEON AURA AR transforms your browser into an interactive AR system where you can:

* 🖐️ Control everything using **hand gestures**
* 🎤 Use **voice commands** to switch modes and abilities
* 🌀 Draw **symbols in the air** to trigger actions
* 🎯 Interact with dynamic targets and **boss mechanics**
* 🎧 Experience **real-time generated audio**
* ✨ See rich, reactive **visual effects**

All running directly in the browser — no hardware, no frameworks, no backend.

---

## 🎮 Core Features

### 🖐️ Hand Tracking (Computer Vision)

* Multi-hand tracking using **MediaPipe**
* Finger-level precision tracking
* Velocity-based interaction system
* Two-hand fusion mechanics

---

### ⚡ Gesture System

* Pinch → fire projectiles
* Hold pinch → charge energy
* Swipe → switch abilities
* Open hand → energy bloom
* Fist → reset blast
* Two hands → fusion core

---

### 🌀 Air Gesture Recognition

Draw shapes in the air to trigger actions:

| Symbol      | Action             |
| ----------- | ------------------ |
| ⭕ Circle    | Portal / Shockwave |
| ➖ Line      | Beam               |
| ⚡ Zigzag    | Lightning          |
| 🔺 Triangle | Shield             |
| ❌ X         | Reset / Destroy    |

---

### 🎤 Voice Commands

Control the system using natural voice:

* `"boss mode"` → start boss battle
* `"shield"` → activate shield ability
* `"lightning"` → switch to lightning spell
* `"galaxy mode"` → change visual mode
* `"clear screen"` → reset visuals

---

### 🎯 Interactive System

* Moving targets (drones)
* Hit detection and physics response
* Score + combo system
* Boss with:

  * health system
  * movement patterns
  * projectile attacks
  * phase transitions

---

### 🎧 Procedural Audio Engine

* Built using **Web Audio API**
* No audio files — everything is generated live
* Audio reacts to:

  * hand position
  * motion speed
  * gesture intensity
  * distance between hands
* Includes filters, delay, reverb, LFO modulation

---

### 🎨 Visual Engine

* Multi-layer canvas rendering
* Particle systems, trails, shockwaves
* Dynamic lighting and glow effects
* Mode-based color systems (10+ modes)
* Screen shake and energy feedback
* Background animation with orbit systems

---

### 🧠 HUD System

Real-time UI displaying:

* FPS, power, charge levels
* Current gesture and spell
* Target tracking system
* Score and combo
* Boss health and status
* Voice + symbol detection

---

## 🛠️ Tech Stack

* **JavaScript (Vanilla)**
* **MediaPipe Hands**
* **Web Audio API**
* **Web Speech API**
* **HTML5 Canvas (multi-layer rendering)**
